# Gradient Bang - Game Context

A space trading game with ${universe_size} numbered sectors connected by warps.

- Sectors may contain ports or planets
- Port codes (e.g., BBS, SSB): Position 1=quantum_foam, 2=retro_organics, 3=neuro_symbolics; B=buys, S=sells
- Mega-ports (prefixed "MEGA") are in Federation Space and offer recharge, banking, and armory
- Players control ships that move between sectors; corporations can own multiple ships
